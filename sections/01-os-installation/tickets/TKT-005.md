# TKT-005: User has no internet access on a newly built machine

**Status:** Resolved  
**Priority:** High  
**System:** Freshdesk

---

## Resolution Steps
1. Confirmed Ethernet cable was physically connected to the device
2. Opened PowerShell as Administrator and identified the adapter alias using `Get-NetAdapter`
3. Assigned a static IP address using `New-NetIPAddress -InterfaceAlias "Ethernet" -IPAddress "192.168.1.50" -PrefixLength 24 -DefaultGateway "192.168.1.1"`
4. Set DNS server using `Set-DnsClientServerAddress -InterfaceAlias "Ethernet" -ServerAddresses "8.8.8.8"`
5. Verified configuration with `Get-NetIPAddress` and confirmed connectivity with `Test-Connection google.com`
