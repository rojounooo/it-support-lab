# TKT-020: User's device not visible to other machines on the network

**Status:** Resolved
**Priority:** Low
**System:** Jira Service Management

---

## Resolution Steps
1. Confirmed network discovery was disabled on the user's device
2. Enabled network discovery via Settings → Network & Internet → Advanced network settings → Advanced sharing settings
3. Asked the reporting colleague to refresh File Explorer → Network and confirmed the device was now visible
4. Confirmed the issue was resolved on both ends
