# TKT-018: Remote worker requires VPN access to connect securely to company resources

**Status:** Resolved
**Priority:** Medium
**System:** Jira Service Management

---

## Resolution Steps
1. Contacted the user and walked them through adding a new VPN connection via Settings → Network & Internet → VPN → Add VPN
2. Provided the connection name, server address, and VPN type for them to enter
3. Advised the user to use their standard company login credentials to authenticate
4. Confirmed the user was able to connect successfully
