# TKT-021: User unable to access external websites — proxy configuration required

**Status:** Resolved
**Priority:** High
**System:** Jira Service Management

---

## Resolution Steps
1. Confirmed the user had no internet access and that the network required a proxy to be configured
2. Obtained the proxy server IP address and port number from the network administrator
3. Configured the proxy via Settings → Network & Internet → Proxy → Manual proxy setup
4. Confirmed the user was able to access external websites after the change
