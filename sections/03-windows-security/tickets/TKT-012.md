# TKT-012: User reports device is running slowly, possible malware infection

**Status:** Resolved
**Priority:** High
**System:** Freshdesk

---

## Resolution Steps
1. Ran a quick scan via Windows Security → Virus & threat protection — no threats detected
2. Ran `Update-MpSignature` to ensure antivirus definitions were current
3. Ran a full scan using `Start-MpScan -ScanType FullScan` to rule out anything the quick scan may have missed
4. Confirmed no threats were found and advised the user on safe browsing practices going forward
