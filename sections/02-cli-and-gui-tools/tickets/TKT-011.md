# TKT-011: User reports application conflicts and system instability — clean boot required to isolate cause

**Status:** Resolved
**Priority:** High
**System:** Freshdesk

---

## Resolution Steps
1. Opened msconfig and navigated to the Services tab
2. Checked Hide all Microsoft services to filter to third-party services only
3. Disabled all remaining third-party services and restarted the device
4. Confirmed system stability after the clean boot
5. Re-enabled services one by one, restarting each time, until the conflicting service was identified
6. Uninstalled the application associated with the conflicting service
